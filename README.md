# Twitter Clone App

Full featured universal (iOS, iPadOS) app.


## Overview
- Written in Swift
- Uses UIKit 
- Designed with MVVM pattern
- Pagination Support for data
- Auto layout Based
- Universal App: Run on iPhone and iPad
- Built in Xcode 15

## Design

The app has five four area:

1. Characters
2. Locations
3. Episodes
4. Settings
5. Search
